# form-driver

An efficient framework for creating forms.

<!-- [使用文档](https://web.hupan.com/m3/index.html) -->

## Setup

``` bash
# 安装依赖
npm run i

# 本地开发
npm run dev

# 本地构建
npm run b

# 发布 tnpm 包
npm run pub
```
